🌱 Smart Agro Portal – Smart Agriculture Information Website

📌 Overview

Smart Agro Portal is a simple and informative web-based project designed to provide essential knowledge about different types of soil and crops. It helps users (especially farmers, students, and agriculture enthusiasts) understand soil characteristics and suitable crops for better agricultural decisions.

🚀 Features

🌾 Information about major crops:
Rice
Wheat
Sugarcane

🌍 Soil type details:
Sandy Soil
Clay Soil
Loamy Soil

• 🖼️ Interactive UI with images and cards

• 📱 Responsive and user-friendly design

• 📄 Separate pages for each soil and crop

🛠️ Technologies Used:

• HTML5 – Structure of the website

• CSS3 – Styling and layout

• Basic UI Design – For visual presentation
